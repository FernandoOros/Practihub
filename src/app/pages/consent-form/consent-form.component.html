Política de privacidad
Declaración de privacidad
Practihub Software Consulting está comprometido con la protección de su privacidad y el desarrollo de tecnología que le brinde una experiencia en línea segura y poderosa. Esta Declaración de privacidad se aplica al sitio web de Practihub Software Consulting y rige la recopilación y el uso de datos. Al usar el sitio web de Practihub Software Consulting, usted acepta las prácticas de datos descritas en esta declaración.
Recopilación de su información personal
Practihub Software Consulting recopila información personal identificable, como su dirección de correo electrónico, entre otros.
También hay información sobre su hardware y software de computadora que es recopilada automáticamente por Practihub Software Consulting. Esta información puede incluir: su dirección IP, tipo de navegador, nombres de dominio, tiempos de acceso y direcciones de sitios web de referencia. Esta información es utilizada por Practihub Software Consulting para la operación del servicio, para mantener la calidad del servicio y para proporcionar estadísticas generales respecto al uso del sitio web de Practihub Software Consulting.
Tenga en cuenta que si divulga directamente información personal identificable o datos personales sensibles a través de las páginas públicas de Facebook de Practihub Software Consulting, esta información puede ser recopilada y utilizada por otros. Nota: Practihub Software Consulting no lee ninguna de sus comunicaciones en línea privadas.
Practihub Software Consulting le anima a revisar las declaraciones de privacidad de los sitios web a los que elige enlazar desde Practihub Software Consulting para que pueda entender cómo esos sitios web recopilan, usan y comparten su información. Practihub Software Consulting no es responsable de las declaraciones de privacidad u otros contenidos en sitios web fuera de Practihub Software Consulting y la familia de sitios web de Practihub Software Consulting.
Uso de su información personal
Practihub Software Consulting recopila y usa su información personal para operar el sitio web de Practihub Software Consulting y entregar los servicios que usted ha solicitado.
Practihub Software Consulting no vende, alquila o arrienda sus listas de clientes o estudiantes a terceros.
Practihub Software Consulting no usa ni divulga información personal sensible, como raza, religión o afiliaciones políticas, sin su consentimiento explícito.
Practihub Software Consulting mantiene un registro de los sitios web y páginas que nuestros clientes visitan dentro de Practihub Software Consulting, para determinar qué servicios de Practihub Software Consulting son los más populares. Estos datos se utilizan para entregar contenido y publicidad personalizados dentro de Practihub Software Consulting a los clientes cuyo comportamiento indica que están interesados en un área temática particular.
Los sitios web de Practihub Software Consulting revelarán su información personal, sin aviso, solo si es requerido por la ley o en la creencia de buena fe de que tal acción es necesaria para: (a) cumplir con los edictos de la ley o cumplir con el proceso legal servido a Practihub Software Consulting o el sitio; (b) proteger y defender los derechos o propiedad de Practihub Software Consulting; y, (c) actuar bajo circunstancias apremiantes para proteger la seguridad personal de los usuarios de Practihub Software Consulting o el público.
Uso de Cookies
Uno de los propósitos principales de las cookies es proporcionar una característica de conveniencia para ahorrarle tiempo. El propósito de una cookie es decirle al servidor web que ha regresado a una página específica. Por ejemplo, si personaliza las páginas de Practihub Software Consulting o se registra en el sitio o servicios de Practihub Software Consulting, una cookie ayuda a Practihub Software Consulting a recordar su información específica en visitas subsecuentes. Esto simplifica el proceso de grabar su información personal, como direcciones de facturación, direcciones de envío, etc. Cuando regrese al mismo sitio web de Practihub Software Consulting, la información que proporcionó anteriormente se puede recuperar, para que pueda usar fácilmente las características de Practihub Software Consulting que personalizó.
Usted tiene la capacidad de aceptar o rechazar cookies. La mayoría de los navegadores web aceptan cookies automáticamente, pero generalmente puede modificar la configuración de su navegador para rechazar cookies si lo prefiere. Si elige rechazar cookies, es posible que no pueda experimentar completamente las características interactivas de los servicios o sitios web de Practihub Software Consulting que visita.
Seguridad de su información personal
Practihub Software Consulting asegura su información personal contra el acceso, uso o divulgación no autorizados. Practihub Software Consulting asegura la información personal identificable que proporciona en servidores de computadoras en un entorno controlado, seguro y protegido contra el acceso, uso o divulgación no autorizados. Cuando se transmite información personal (como un número de tarjeta de crédito) a otros sitios web, se protege mediante el uso de cifrado, como el protocolo Secure Socket Layer (SSL).
